 Agenda em C

Projeto de uma agenda simples desenvolvida em linguagem C, com foco em estudo de
lógica de programação, uso de `struct` e manipulação de arquivos.

##  Funcionalidades
- Adicionar contatos
- Listar contatos cadastrados
- Buscar contato pelo nome
- Persistência de dados em arquivo texto

##  Tecnologias utilizadas
- Linguagem C
- GCC
- Terminal / CMD

##  Como compilar e executar

```bash
gcc agenda.c -o agenda
./agenda
